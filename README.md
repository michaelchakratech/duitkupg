# Duitku.NET

[![NuGet Version](https://img.shields.io/nuget/v/YourPackageName.svg)](https://www.nuget.org/packages/YourPackageName/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A lightweight, asynchronous C# client library for integrating the **Duitku Payment Gateway** into .NET applications.

This library is unofficial version, use it at your own risk.

## 🚀 Features

* **Easy Integration**: Simple DI-based setup.
* **Async First**: Fully asynchronous API calls for better performance.
* **Type Safe**: Strongly typed request and response objects.
* **Comprehensive**: Supports Inquiry, Callback handling, and Check Transaction status.

## 📦 Installation

Install via NuGet Package Manager:

```bash
dotnet add package Duitku.NET
